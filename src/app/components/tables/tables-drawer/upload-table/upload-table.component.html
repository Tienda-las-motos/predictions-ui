Puedes subir un archivo CSV con el registro de tus ventas que contenga las siguientes columnas:
  • Fecha: Fecha en la que se realizó la venta.
  • Codigo: El código de identificación del artículo vendido
  • Descripcion: Nombre o referencia del artículo vendido
  • Unidades: Cantidad de unidades vendidas.
  • Unitario Venta: precio unitario de venta
  • Ventas: Total de las ventas de esa transacción
  • Costo Unitario: precio unitario de costo de compra
  • Total Costo: Total de costo de las unidades vendidas
  • Margen: Diferencia entre el valor de las ventas menos el total del costo
  • PorMargen: El margen de ganancia
